# angelcloud-master
## Angelive博客后端

效果展示: [Angelive](http://angelive.fun) 


## 项目介绍
    核心框架: Spring Cloud 2.0
    持久层框架: Mybatis Plus 3.0
    数据库: MySQL5.7 Redis
    数据库连接池: HikariCP
    安全框架: Spring Cloud Oauth2 (暂未开发)
    搜索引擎: ElasticSearch 7.2 (暂未开发)
    日志管理: Slf4J
    第三方服务: 七牛云
